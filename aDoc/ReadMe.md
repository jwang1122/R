## Hello World

```R
# Print Hello world.
print("Hello World")

print(23.5 + 11.3)
myString <- "Hello, World!" # <- assignment operator
print(myString)
```
* save the code to hello.r
```
rscript hello.r
```
* Output will be
```
C:\Users\12818\workspace\R>Rscript hello.r
[1] "Hello World"
[1] 34.8
[1] "Hello, World!"
```
